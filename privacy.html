Privacy Policy

FreeOTP does not collect any data from your mobile device.

  • Camera access is used only for scanning OTP QR codes.
  • Internet access is used only for fetching token icons.
  • Bluetooth is used only to transfer token codes to remote systems.
  • Location access is only used when required by Bluetooth (Android).

If you believe this policy has been violated in any way, please file an issue.
